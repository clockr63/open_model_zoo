# [Tizen Vision](https://developer.tizen.org/development) - Open Model Zoo repository
[![Apache License Version 2.0](https://img.shields.io/badge/license-Apache_2.0-green.svg)](LICENSE)

This repository includes reference models and the description of thier architectures which can be supported by Tizen Vision API. Use this free pre-trained models to speed-up the develpment or train your own models to run applications for which models in the repository are not suitable.

## How to Contribute
We welcome contributions to the Open Model Zoo repository. We will review your contribution and, if any additional fixes or modifications are needed, may give you feedback to guide you. When accepted, your pull request will be merged into the GitHub* repositories.

By contributing to the project, you agree to the license and copyright terms therein [Apache License Version 2.0](LICENSE) and release your contribution under these terms.
